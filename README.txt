Excel to SQLite Converter
=========================

Quick Start:
1. Put Excel files in the 'input' folder
2. Double-click 'run.bat' to start conversion
3. Check results in the 'output' folder

Requirements:
- Python 3.7+
- Install dependencies: pip install -r requirements.txt

Features:
- Batch convert multiple Excel files
- Preserve original data integrity
- Generate detailed reports
- Support .xlsx and .xls formats

View Database:
Use online SQLite viewer: https://sqliteviewer.app/
Drag .db files from output folder to the website.
